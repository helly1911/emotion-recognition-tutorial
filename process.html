Process

System Architecture

Overall Process is divided into three subparts.

Face Detection

Face detection finds the human face in the input image. And if the input is a video, then Face detection is only performed on keyframes and a tracking algorithm is applied on interval frames.

Feature Extraction

After face detection, the next task is to perform feature extraction. Geometric based feature approach is used for extracting face features. Face landmarks are crucial for any facial expression. So, It would be performed by detecting specific landmarks on the face such as eyebrows, eyes, nose, mouth, etc. and then perform math by some kind of algorithms which finally able to detect a personsâ€™ emotions.

Feature classification

For, emotion detection system, here, we classify seven universal emotions such as happy, sad, anger, disgust, fear, surprise, and neutral. So, after feature extraction, using machine learning algorithms, we able to classify them into seven classifications.
